## 前台分类建议

- `CT 影像系统`
- `磁共振系统`
- `X 线与超声`
- `冠脉支架`
- `球囊与导管`

这套分类适合首版官网：

- 数量不多，但足够区分采购场景。
- 前台筛选不会太复杂。
- 后续还能继续扩到 DSA、乳腺机、导丝、导引导管等子类。

## 产品详情字段建议

### 所有产品统一字段

- 正式名称（中 / 英）
- 厂商名称（中 / 英）
- 型号
- 简介
- 适用场景
- 关键规格
- 交付与资料

### 影像设备推荐额外信息写法

- 适用场景：
  - 科室定位
  - 临床用途
  - 适合医院层级
- 关键规格：
  - 层数 / 场强 / 通道 / 旋转速度 / 分辨率 / 平台特性
- 交付与资料：
  - 产品页
  - 宣传册
  - 安装 / 机房 / 培训说明

### 介入耗材推荐额外信息写法

- 适用场景：
  - 适应证
  - 手术环节
  - 配套使用位置
- 关键规格：
  - 直径范围
  - 长度范围
  - 兼容导丝 / 导管
  - 载药量 / 额定压力 / 平台特性
- 交付与资料：
  - IFU
  - Brochure
  - 受控注册资料

## 资料中心规则

### 公开资料

- 官方产品页
- 官方公开 brochure
- 对外彩页

### 申请后获取

- IFU
- 注册资料
- 招投标资料
- 受控技术包

## 图片策略

- 只有在拿到可用官方图或授权图时，才直接显示产品图。
- 没有真实可用图片时，前台用型号占位卡，不继续使用无关的通用医疗图。
- 这样能保证“信息真实”，也避免误导。
